1 - Cliente: É uma aplicação que está conectada à internet. Sua principal função é receber a interação do usuário e traduzi-la em uma requisição para outro computador chamado Servidor Web.
O computador(doméstico) é um Cliente, todo computador tem um IP único, que podem ser identificados através dele.

2 - Servidor: O servidor tambem é um computador, só que possui um software que irá responder as requisições recebidas do Cliente.
A principal função do servidor é armazenar,processar e entregar páginas web para os Clientes. No caso do www.google.com que é o servidor de busca, é um Web Server.

3 - Endereço IP: É um identificador numérico para um dispositivo conectado a rede TCP/IP.  Todo dispositivo conectado a internet tem um endereço IP unico que irá identifica-lo, e será usado para comunicar com outros dispostivos.

4 - TCP/IP: É o principal protocolo de envio e recebimento de dados na internet. O TCP será responsável por definir como diferentes aplicações se comunicam na rede e tambem em como os dados enviados serão divididos em pacotes.
O IP, define como os dados divididos em pacotes devem ser enviados para chegar corretamente ao destinatário. Ou seja, o TCP/IP é a união dos protocolos TCP e IP, uma pilha de protocolos que é dividida em quatro camadas: aplicação, transporte, rede e interface.

5 - ISP: É o provedor de Internet. É qualquer empresa que nos fornece acesso ao mundo Web. É a parti deles que conseguimos ser conectados á Internet.

6 - DNS: É um dispositivo com uma base de dados distribuída que gerencia os nomes de serviços, computadores ou qualquer dispositivo conectado a internet.
Ele relaciona o endereço nominal(google.com) com o endereço IP. São eles que localizam e traduzem para números (um endereço IP) as buscas por sites que digitamos no navegador.

7 - Port Number: É uma forma de indentificar um processo especifíco encaminhado para o servidor.

8 - Host: É qualquer computador conectado à rede, seja como cliente,servidor ou qualquer outro tipo de dispostivo. Cada host tem um endereço IP único. Cada host tem um endereço IP único. Para o site que estamos buscando, o host é o servidor web que hospeda a página na web.
Host e Servidor são diferentes, o Servidor é um tipo de host - uma maquina específica com endereço IP. Por outro lado, o host pode ser também uma organização inteira que fornece um serviço de hospedagem para manter inúmeros servidores web.

9 - HTTP - É o protocolo utilizado para a comunicação entre navegadores e servidores na Internet.

10 - URL: é o indenticador de um recurso na web especifíco. O URL identifica o protocolo que será utilizado para comunicarmos com o servidor (https), o nome do host(google) e o recurso que querendo acessar(fotosdecoelho).
